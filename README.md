### 2. プログラムを作る


ローカルで適当にブランチを切る

```
## わかりやすいように、自身の名前でブランチを切ってください。
git checkout -b hiroyuki.takamura
```

適宜プログラムを作成して、コミットする

pushする
```
git push origin hiroyuki.takamura
```

Github上で、プルリクエストを作成する
